Detailed Description

A simple holder for the POJO to serialize via FastJsonHttpMessageConverter along with further serialization instructions to be passed in to the converter.

On the server side this wrapper is added with a

ResponseBodyInterceptor

after content negotiation selects the converter to use but before the write.

On the client side, simply wrap the POJO and pass it in to the

RestTemplate

.
